# Project Atlas 
Project Atlas is a personal learning accountability tracker. 

## Problem 
I struggle to maintain a consistent study schedule when I do not have external structure, deadlines, or accountability.

## Goal
Build a tool that helps me plan daily learning tasks, track completion, and analyze my study habits over time.

## Version 1 Features 
-Display today's tasks
-Mark tasks as complete
-Calculate daily completion percentage 
-Save progress between sessions
